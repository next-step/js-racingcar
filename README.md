### 🎯 step1

- [x] 자동차 이름을 쉼표를 기준으로 입력한다. 이름은 5자 이하만 가능하다.
- [x] 경주를 시작할 때 자동차의 이름을 보여준다.
- [x] 사용자는 자동차를 몇 번 이동시킬 것인지 시도할 횟수를 입력한다.
- [x] 주어진 횟수만큼 자동차들은 전진 혹은 멈춤을 할 수 있다.
- [x] 각 자동차가 전진하는 조건은 0~9 사이의 랜덤값을 구했을 때 4이상일 때이다. 3이하면 멈춘다
