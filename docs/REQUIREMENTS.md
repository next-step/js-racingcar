## 자동차
- [x] 자동차는 5자 이하의 이름을 갖는다.
- [x] 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우, 전진한다

## 입출력
- [x] 쉼표를 기준으로 자동차 이름을 받는다.
- [x] 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
- [ ] 우승자가 여러명일 경우, 쉼표로 구분하여 출력한다.
- [ ] 사용자가 잘못된 입력 값을 작성한 경우 프로그램을 종료한다.

## 경주
- [x] 5회 반복된다.
- [ ] 우승자를 발표한다.
